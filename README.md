# E-cart-Backend
This project is a backend system for an e-commerce application (E-Cart) built using Java, JDBC, and SQL. It provides core functionalities required to manage an online shopping platform, including user management, product handling, cart operations, and order processing.

🚀 Features

👤 User registration and authentication

📦 Product management (Add, Update, Delete, View)

🛒 Shopping cart functionality

📑 Order placement and tracking

🗄️ Database integration using JDBC

🔐 Basic input validation and error handling

🛠️ Tech Stack

Java (Core Java)

JDBC (Java Database Connectivity)

SQL (MySQL / any relational database)

⚙️ Setup Instructions

Clone the repository

Configure database credentials in the connection file

Create required tables using provided SQL scripts

Compile and run the project
